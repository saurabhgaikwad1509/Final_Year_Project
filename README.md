# Food Ordering Web Application

This is a Food Ordering Web Application developed as a Final Year Project.
Users can browse food items, add them to cart, and place orders online.
The project demonstrates full-stack web development concepts.

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Bootstrap
- XAMPP

## How to Run the Project
1. Install XAMPP
2. Start Apache and MySQL
3. Copy the project folder to `htdocs`
4. Import database from `database/food.sql`
5. Open browser and go to:
   http://localhost/FOOD

## Database
- Database Name: food_db
- Tables: users, orders, food_items, admin
